| [1][id1]  | [Оптимизация затрат на энергоресурсы при производстве металла][id1] | Имеются данные полученные от заказчика, описывающие процесс обработки металла. Необходимо построить модель предсказывающую финальную температуру металла с минимально возможным значением MAE.
Эффект от модели - оптимизация расходов на производственные затраты энергии. В качестве одного из способов снижения затрат является поддержание финальной температуры изготавливаемого материала не выше требуемых значений. В связи с этим требуется разработь модель, которая на основании входных данных в виде температуры, мощности электродов, количества засыпных и проволочных материаллов и т.п. предсказывала бы финальную температуру металла. | Scikit-learn, LightGBM, Tensorflow, Pandas, NumPy, Matplotlib, Seaborn. Машинное обучение, предобработка данных, визуализация. |

| [5][id5]  | [Предсказание коэффициента восстановления золота из золотосодержащей руды][id5] | Имеются данные с различными признаками, зафиксированными в процессе нескольких этапов очистки золотосодержащей руды. Качество модели оценивается метрикой smape, которую необходимо минимизировать. | Scikit-learn, Pandas, NumPy, SciPy, Matplotlib, Seaborn.
